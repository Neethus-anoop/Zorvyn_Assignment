This project implements a backend system for a finance dashboard using Node.js and Express.js, focusing on clean architecture, data management, and access control. It includes user and role management with role-based permissions, secure authentication using JWT, and full CRUD operations for financial records such as income and expenses. The system also provides dashboard APIs for aggregated insights like total income, expenses, and balance. Data is stored using SQLite, and the project follows an organized MVC structure with basic validation, error handling, and Swagger-based API documentation, ensuring a maintainable and scalable backend solution
